## 📖 درباره پروژه

این پروژه وبسایت تک‌صفحه‌ای (**Single Page**) مؤسسه فرهنگی قرآن و عطرت حافظان یاس نبی(س) است که از سال **۱۳۸۸** در شهر **خرم‌آباد** فعالیت دارد.

سایت با طراحی مدرن، انیمیشن‌های روان و رابط کاربری زیبا ساخته شده و بدون نیاز به هیچ فریمورک یا کتابخانه‌ای کار می‌کند.

## ✨ ویژگی‌ها

| ویژگی | توضیح |
|---|---|
| 🎨 **طراحی مدرن** | رابط کاربری با ترکیب رنگ سبز زمردی و طلایی |
| 📱 **واکنش‌گرا** | نمایش عالی در موبایل، تبلت و دسکتاپ |

| 🗺️ **نقشه تعاملی** | نمایش موقعیت مؤسسه روی نقشه گوگل |
| 📊 **شمارنده متحرک** | نمایش آمار با انیمیشن شمارش |
| 🖼️ **گالری تصاویر** | لایت‌باکس برای نمایش تصاویر بزرگ |

| 🧭 **منوی هوشمند** | هایلایت خودکار بخش فعال در منو |

## 🏫 بخش‌های سایت

- **صفحه اصلی** — بنر ورودی با انیمیشن ذرات و افکت شفق قطبی
- **درباره مؤسسه** — معرفی مؤسسه و مدیریت خانم یاسمیان
- **آمار** — تعداد قرآن‌آموزان، اساتید، حافظان و سال‌های فعالیت
- **دوره‌های آموزشی** — مهد قرآن، پیش‌دبستانی، روخوانی، چرتکه، ریاضی، نقاشی
- **گالری** — تصاویر واقعی از فضا و فعالیت‌های مؤسسه
- **اساتید** — معرفی تیم آموزشی مؤسسه
- **تماس** — آدرس، تلفن و نقشه تعاملی

## 🚀 راه‌اندازی

فقط کافیه فایل `index.html` رو توی مرورگر باز کنی:

```bash
# کلون کردن پروژه
git clone https://github.com/7nimor/hafezanyasenabi.git

# باز کردن در مرورگر
cd hafezanyasenabi
open index.html
```

> هیچ نصب یا build خاصی لازم نیست! 🎉
